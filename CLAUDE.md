# CLAUDE.md - City of Asylum PDX

## Development Workflow

**IMPORTANT**: Always use local development server before deploying to Netlify to save deploy credits.

### Local Development Process:
1. Start local server: `npm start` (runs on http://localhost:8080)
2. Make changes and test locally
3. Only push to Git when ready to deploy to production

### Design Guidelines

**Typography:**
- Hero section: Use sans-serif font (not serif)
- Body content: sans-serif

**Color Patterns:**
- Navigation: White background with green text
- Hero: Solid background (no gradients) - match other section patterns
- Sections alternate: cream, sage/10, cream, etc.

**Current Color Palette:**
- forest-dark: #1a3a2e
- forest: #2d5f4c
- forest-light: #4a8b6f
- sage: #8fb399
- cream: #f5f3ed
- earth: #8b7355
